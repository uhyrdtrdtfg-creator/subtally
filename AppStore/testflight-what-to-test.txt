【What to Test — 1.0 首发】

核心验证流程:

1. 添加订阅
   - 从模板库挑 2-3 个常见服务(Netflix / ChatGPT / iCloud+)
   - 手动添加一个冷门服务,试试金额、周期、分类
   - 开启"免费试用"字段,验证倒计时

2. iCloud 同步
   - 登录同一 Apple ID 在第二台设备打开 App,等 10-30s 数据应自动出现
   - 在设备 A 改价格,设备 B 应该在 1 分钟内刷新

3. 提醒通知
   - 把一条订阅的下次扣款日期设为明天,开启提醒
   - 锁屏 / 通知中心应在到期前 2 天与当日收到本地推送

4. 桌面小组件
   - 长按桌面 → + → 搜索"订算"
   - 小/中/大号小组件在主屏与锁屏都能加
   - Widget 应显示本月总支出或最近 3 条即将到期

5. 灵动岛 / Live Activity(需真机 iPhone 14 Pro+)
   - 添加一条试用中的订阅,试用结束时间在 48h 内
   - 应在锁屏与灵动岛看到倒计时胶囊

6. Vision OCR 扫账单
   - 我的 → 扫描支付凭证 → 选一张支付宝/微信账单截图
   - 应自动识别金额、日期、商家,一键套用品牌图标

7. Webhook(可选,需要你有 Slack/Bark/Telegram endpoint)
   - 设置 → 自动化 → 新建 → 选 Slack 预设
   - 填 URL,保存
   - 新建或修改一条订阅,应在 Slack 收到消息

8. 年度分享卡片
   - 我的 → 年度分享卡片 → 应看到 1080×1920 预览,含 Top 服务 / Top 分类 / 四宫格统计
   - 点分享应弹出系统分享 sheet,保存到相册可落盘

反馈优先级:
- P0:崩溃、数据丢失、iCloud 同步失败、支付扣款金额显示错误
- P1:UI 错位、暗/亮色切换不顺、某个小组件渲染空白
- P2:文案优化、布局审美建议、额外币种/模板

请提供截图与机型 + iOS 版本(设置 → 通用 → 关于本机)。
